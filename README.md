# Luxury-Housing-Sales-Analysis-Bengaluru
This Analysis contains Luxury housing sales in Bengaluru by using python, SQL and PwerBI.
In this Analysis, I have first processed the Data cleaning with the help Python. 
I have handled the Null values, Incorrect format and derived few new columns for visualization.
And loaded the cleaned CSV file into SQL.
Imported the data from SQL to PwerBI and answered the visualization questions.
